# star-rating

原生js实现星级评分

可传入分数展示星级评分，不可编辑

可传入阐述设置星级评分，可编辑

点击会返回选中星星个数

支持半颗星星的这只和整棵星星的设置

如果只是展示，传入的分数有小数，则展示半颗出来，例如3.2和3.5，都会显示3棵半的星星

没有小数展示整数棵星星出来，例如：4就展示4棵整数星星
